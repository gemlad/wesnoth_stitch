# Sprint reviews

Each review is a **single self-contained HTML file** — no external CSS, fonts, scripts or
images (screenshots are inlined as data URIs). Open one in a browser directly from a
checkout: no build step, no network, no server. It works offline and it will still work in
five years.

| Review | Covers |
| --- | --- |
| [sprint-review-m2-m3.html](sprint-review-m2-m3.html) | **Milestones 2 & 3** — the conversion pipeline (DMC mapping, colour reduction, the symbol set and the cap it implies) and the export (PNG, and the printable PDF chart). Also: what the measurements settled, and the decisions still waiting on a human. |

Milestone 1's review exists only as a published artifact and predates this folder.

## Why these are checked in

A review is worth more later than on the day it is given — it is the record of *why* a
decision went the way it did, next to the numbers that decided it. A link to a hosted page
is not that record: it needs an account, a network, and a service that still exists.
The file is.
